# Amazon Sales Optimization & Profitability Intelligence

## 📌 Project Overview

This project focuses on analyzing Amazon sales data to identify profit-draining products, optimize fulfillment efficiency, and generate actionable business insights using SQL, Python, and Power BI.

---

## 🎯 Business Problem

A mid-sized retailer selling on Amazon is experiencing reduced profitability due to high return rates, shipping costs, and order cancellations. The goal is to identify the major factors affecting profitability and recommend improvements.

---

## 🎯 Objectives

- Analyze sales performance
- Identify high-return products
- Measure profitability
- Analyze fulfillment efficiency
- Build an interactive Power BI dashboard
- Generate business recommendations

---
